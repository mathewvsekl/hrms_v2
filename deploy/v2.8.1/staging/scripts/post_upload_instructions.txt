# HRMS v2.8.1 Post-Upload Instructions (DirectAdmin)

1. UPLOAD
   - Upload 'HRMS_V2_v2.8.1_FULL.zip' one level ABOVE your public_html folders.

2. EXTRACT
   - Extract the ZIP. This will populate:
     - public_html/ (Main source - v2.8.1)
     - database/ (Migration scripts)
     - scripts/ (Deployment guides)
     - storage/ (Internal files)
     - .env.example
     - VERSION

3. INITIALIZE DATABASE
   - Via phpMyAdmin, import 'database/schema.sql'.
   - Then apply 'database/migration_v2.8.1.sql' if updating from v2.7.7.

4. SETUP config/config.php
   - In 'public_html/config/', ensure 'config.php' matches your production environment.

5. VERIFY
   - Access your domain to log in.
